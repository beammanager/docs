## Database setup

### Database installation

```bash tab="Ubuntu/Debian"
sudo apt install mariadb-server mariadb-client
```

```bash tab="Fedora"
sudo dnf install mariadb-server mariadb
```

```bash tab="openSUSE"
sudo zypper install mariadb mariadb-client
```
